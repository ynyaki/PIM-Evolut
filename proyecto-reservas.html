Proyecto Gestión de Reservas

Base de Datos

El uso de una base de datos es crucial para manejar grandes volúmenes de datos de manera eficaz y escalable.

Frontend Web

Una interfaz en línea para hacer uso de la aplicación de manera sencilla, para cualquier usuario.

CRUD con Java

El backend o parte trasera será un sistema de llamadas a la base de datos desde Java, para acceder a la información y entregar la que necesite el usuario.

Objetivo

El objetivo del proyecto es crear un sistema de reservas (de cualquier tipo) para manejar todas las disciplinas en combinación. Completa la formación y prepara al estudiante para los proyectos reales profesionales.
